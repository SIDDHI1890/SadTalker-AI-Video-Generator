# 🎭 SadTalker AI Video Generator

An AI-powered application that generates realistic talking face videos from a single image and audio input using the SadTalker model. This project performs audio-driven facial animation, lip synchronization, and expression mapping to create natural-looking videos.

---

## 🚀 Features

- 🖼️ Convert a static image into a talking face video  
- 🎤 Audio-driven lip-sync generation  
- 😀 Realistic facial expression animation  
- 🌐 Interactive web interface using Gradio  
- ⚡ Supports CPU (GPU recommended for faster performance)  

---

 Tech Stack

- **Programming Language:** Python  
- **Frameworks/Libraries:** PyTorch, Gradio  
- **Model:** SadTalker  
- **Other Tools:** FFmpeg
